================================================================================
KEY DESIGN DECISIONS
================================================================================

1. PYTORCH FRAMEWORK
   - Better for custom training loops
   - More flexible than TensorFlow
   - Excellent for hackathon rapid prototyping

2. U-NET ARCHITECTURE
   - Industry standard for image denoising/segmentation
   - Skip connections preserve fine details
   - ~1.9M parameters (trainable in hours)

3. SKIP CONNECTIONS
   - Pass encoder features directly to decoder
   - Prevents information bottleneck
   - Critical for image denoising

4. COMBINED LOSS (L1 + MSE)
   - L1 loss: robust to outliers
   - MSE loss: smooth gradient flow
   - Better convergence than single loss

5. NUMPY ARRAY STORAGE
   - As requested, all data kept as numpy arrays
   - Easy to integrate with any framework
   - No library lock-in
